## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - all projects run client-side!

### Running the Projects

#### Option 1: Clone Repository
```bash
git clone https://github.com/lieujoy/InternPE.git
cd InternPE
```

#### Option 2: Download ZIP
Download and extract from GitHub

### Open Projects
1. Navigate to desired project folder:
   - `docs/01Calculator/` for Calculator
   - `docs/02TenVault/` for TEN VAULT Store
   - `docs/03JustDoIT/` for Todo App
2. Open `index.html` in your web browser
3. Start exploring!
