## 🚀 Getting Started

### Prerequisites
- Modern web browser (Chrome, Firefox, Safari, Edge)
- No server required - all projects run client-side!

### Running the Projects

1. **Clone the repository**:
   ```bash
   git clone https://github.com/lieujoy/InternPE.git
   cd InternPE
   ```

2. **Open any project**:
   - Navigate to the project folder (e.g., `docs/01Calculator/` or `docs/02TenVault/`)
   - Open `index.html` in your web browser
   - Start exploring!
